# BookFlow — Онлайн библиотека и трекер чтения

## Описание проекта

BookFlow — это веб-приложение для поиска, сохранения и организации книг с использованием Google Books API.

Пользователь может:

* искать книги по названию, автору или жанру
* просматривать подробную информацию о книге
* сохранять книги в личную библиотеку
* создавать списки чтения
* отслеживать статус чтения
* ставить оценки и писать отзывы
* получать персональные рекомендации

Проект решает проблему удобной организации чтения и хранения информации о книгах в одном месте.

Целевая аудитория:

* люди, которые читают книги регулярно
* студенты
* любители литературы
* пользователи, которые хотят вести личный reading tracker

---

#  Основная идея проекта

Google Books API предоставляет огромную базу книг, однако пользовательский опыт ограничен:

* отсутствует удобная система организации чтения
* нет персональной библиотеки
* нет tracking-системы прогресса чтения
* отсутствует современный UI/UX
* нет удобной системы заметок и списков

BookFlow решает эту проблему, предоставляя современный интерфейс и дополнительные возможности поверх Google Books API.

#  User Stories

## Для пользователя

* Как пользователь, я хочу искать книги по названию, чтобы быстро находить нужную литературу.
* Как пользователь, я хочу сохранять книги в избранное, чтобы вернуться к ним позже.
* Как пользователь, я хочу менять статус чтения книги, чтобы отслеживать свой прогресс.
* Как пользователь, я хочу оставлять отзывы, чтобы делиться мнением.
* Как пользователь, я хочу получать рекомендации, чтобы находить новые книги.

## Для администратора

* Как admin, я хочу модерировать отзывы, чтобы поддерживать качество контента.
* Как admin, я хочу управлять пользователями, чтобы контролировать систему.
